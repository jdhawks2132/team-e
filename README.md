### Project Management Application (E-ORG)

- User Can:
  - Sign Up or Login Sign out
    - User
      - Email/ PW/ Icon
    - Projects (FULL CRUD)
      - Titles
      - Description
      - Due Dates
      - Team Members
  - Click on Projects for more info
  - Chat with Project Team
  - Assign existing users to projects
